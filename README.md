# Тестовое задание — лендинг курса «Петербург сквозь года»

Верстка одностраничного сайта по макету. Bootstrap 4, jQuery 3.

## Использованные технологии

- Bootstrap 4 для адаптивки
- jQuery для плавного скролла
- Шрифты: BERNIERDISTRESSED (локальный), PT Sans (Google Fonts)

## Структура

- `index.html` — основная разметка
- `css/style.css` — стили
- `js/main.js` — скрипт для скролла
- `img/` — логотип
- `Тестовое на верстальщика Mailiq/` — исходные файлы (шрифт, картинки)

## Запуск

Просто открыть `index.html` в браузере. Все библиотеки подключаются через CDN.

## Реализовано

- Фиксированное меню сверху
- Плавный скролл по якорям
- Адаптивная верстка (Bootstrap grid)
- Аккордеон в FAQ
- Бургер-меню на мобильных
